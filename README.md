# alfred-reminder-chinese

自动添加提醒事项的 alfred 插件，支持中文时间自动识别。

### ➡️ 安装

下载项目中的 [**快速添加提醒事项.workflow**](https://github.com/taohonghui/alfred-reminder-chinese/blob/main/快速添加提醒事项.alfredworkflow) ，双击打开，自动导入 alfred

### ➡️ 示例

```
t 记录一些有趣到事情
t 今天 把球鞋买了
t 明天 录制歌曲
t 下周四18点 开大会
t 在5分钟内 完成整理
t 在2小时内 处理要做到事情
t 在3天2小时 内解决bug
t 22/12/20 去看电影
t 下午两点 记录工作
```
